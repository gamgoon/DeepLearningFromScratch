### numpy

- np.ndim() : 배열의 차원 수
- shape : 배열의 형상
- 행렬 : 2차원 배열, 배열의 가로방향을 행row, 세로 방향을 열column.
- np.dot() : 행렬 곱(행렬의 내적), 피연산자의 순서가 다르면 결과도 다르다.
    - 형상이 다른 행렬의 내적은 행렬 A의 1번째 차원의 원소 수(열 수)와 행렬 B의 0번째 차원의 원소 수(행 수)가 같아야한다.
